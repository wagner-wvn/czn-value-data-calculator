# Chaos Data Value Calculator  
A community-powered tool that helps Chaos Zero Nightmare players calculate their **Save Data (Data Value)** safely during Chaos runs.

### What this tool does
- Calculates the exact cost of each action (removals, copies, epiphanies, elites, etc.)
- Helps players avoid exceeding their tier limit
- Makes Chaos runs easier to plan, safer, and more consistent
